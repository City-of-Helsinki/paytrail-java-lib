Unofficial Paytrail Java Library
=

See documentation at https://docs.paytrail.com/#/

# Install

#### Example for maven:
```
  <dependencies>
    <dependency> 
      <groupId>org.helsinki</groupId>
      <artifactId>paytrail-java-lib-core/artifactId>
      <version>1.0.1-SNAPSHOT</version>
    </dependency> 
  </dependencies>
```
